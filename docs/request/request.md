# Request

①settings/setting_window.pyに設定できる項目として以下のフォント関係の項目を追加してください。
setting_jsonにも項目を追加し、字幕を設定する処理に対してこれを参照するように修正してください。
また、フォント関係の項目をまとめるために、現在の設定画面にタブを追加して、「一般」と「字幕」に分けてください。
- テロップ ON/OFF
- 文字色（例：#ED1C24）
- フォントの大きさ（数字）
- フォント種類(例：Yu Gothic UI)
- 1行最大文字数(数字)
- アウトライン(縁取り)の色（例：&H00000000）
- アウトラインの太さ（数字）
- 背景/影の色（例：&H64000000）
- 太字（チェックボックス）
- 斜体（チェックボックス）
- 下線（チェックボックス）
- 打消し線（チェックボックス）
- 文字間隔（数字）
- 回転角度（数字）
- 縁取りスタイル（ドロップダウン）
    value - text
    1 - 縁取り + ドロップシャドウ
    3 - 不透明ボックス
- 表示位置（ドロップダウン）
    value - text
    1 - 下段（左）
    2 - 下段（中央）
    3 - 下段（右）
    4 - 中段（左）
    5 - 中段（中央）
    6 - 中段（右）
    7 - 上段（左）
    8 - 上段（中央）
    9 - 上段（右）

- 余白
    左（数字）
    右（数字）
    下（数字）

②設定画面に「開始」ボタンを設置する
このボタンはmain.pyを実行する。
