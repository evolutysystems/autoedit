# Request

- フォントのプレビュー画面が欲しい
    現在設定しているフォント、設定したフォントのプレビューを画面上に置くことができないか。

    settings_window.pyでフォントを追加できるようにする。対応できるフォントファイルのみ対応する。

    フォントサイズは小さめのテキストボックス。
