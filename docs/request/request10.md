# Request
現状settings_window.pyでテロップの色は１色しか選択するとこができませんが、これを３色にしてください。
- 配信者カラー(現状の色)
- サブカラー
- コメントカラー

(追加2)
settings_window.pyアウトラインカラーもそれぞれ３つ作成してください。
- 配信者アウトラインカラー
- サブアウトラインカラー
- コメントアウトラインカラー

テロップ一覧画面の項目で、「使用」右隣に以下の項目を追加する。また、一覧にはチェックボックス(もしくはラジオボタン)を置く。
- 配信者
- サブ
- コメント

上記の項目は１行につき１つしか選択することができないようにする。
選択した項目のカラーをテロップに適応する。
- 配信者 = 配信者カラー・配信者アウトラインカラー
- サブ = サブカラー・サブアウトラインカラー
- コメント = コメントカラー・コメントアウトラインカラー

(追加)
一覧で「コメント」を選択した場合、テロップ出力時にテロップの戦闘に「コメント：」と記載し、改行する。
