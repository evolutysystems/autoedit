# Request
現状の問題点を以下に示す。
- 誤訳が多い。
これを改善するために以下の修正を行います。
- 字幕編集画面を追加します。
- パイプラインの「フルテロップ」後に、生成したテロップを一覧上に並べた画面を開く。
- 項目は以下。
    ・時間
    ・字幕
    ・使用
- 「時間」には動画のどの部分なのか時間を表示する
- 「字幕」にはその時間に表示する字幕を表示する。編集可能。
- 「使用」にはチェックボックスを配置する。その字幕を動画として使用するかの判断をチェックボックスで判定する。初期値は一覧上の全てをチェックした状態にする。
- 「字幕決定」ボタンを配置する。ボタンを押下すると続きのパイプラインを続行する。
