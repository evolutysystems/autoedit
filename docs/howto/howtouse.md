# 切り抜き自動編集（AutoEdit）操作マニュアル

動画を1本選んで「実行」を押すだけで、**無音カット → フルテロップ生成 → オープニング／エンディング結合 → 書き出し**までを自動で行うアプリです。
このマニュアルは、初めて使う方がインストールから動画の書き出しまで迷わず進められるように、画面に沿って手順を説明します。

対象OS: Windows 11

## 1. インストール

1. 配布サイトから **`AutoEditSetup.exe`** をダウンロードします。
2. ダウンロードした `AutoEditSetup.exe` をダブルクリックします。
3. 画面の案内に沿って進みます。途中で次の2点を選びます。
   - **編集した動画の出力先フォルダ** … 完成した動画を保存する場所を選びます（後から設定画面でも変更できます）。
   - **デスクトップにショートカットを作成する** … 必要ならチェックを入れます。
4. インストーラーが本体を自動でダウンロード・展開します。しばらく待つと完了します。

> **メモ**
> - インストールに管理者権限は不要です（ユーザーフォルダにインストールされます）。
> - 動画変換に必要な FFmpeg はアプリに同梱されているため、別途インストールは不要です。
> - インターネット接続が必要です（本体のダウンロードに使用します）。

---

## 2. アプリを起動する

- **デスクトップのショートカット**、または **スタートメニューの「AutoEdit」** から起動します。
- 起動すると「**切り抜き自動編集 実行**」というウィンドウが開きます。これがメイン画面です。

メイン画面には次のものがあります。

| 項目 | 役割 |
|---|---|
| 入力動画 | これから編集する動画ファイルのパスを表示・入力します。**動画ファイルをここへドラッグ&ドロップしても選べます** |
| 参照... | 編集したい動画をファイル選択ダイアログから選びます |
| 設定... | 設定画面を開きます（初回は先にここを確認するのがおすすめ） |
| 実行 | 編集処理を開始します |
| 進捗バー／ステータス | 処理の進み具合を表示します |

---

## 3. はじめに設定を確認する

初めて使うときは、メイン画面の **「設定...」** ボタンを押して、内容を確認しておきましょう。
設定画面は「**一般**」タブと「**字幕**」タブに分かれています。変更したら必ず下部の **「保存」** ボタンを押してください。

### まず確認したい項目

- **一般タブ → 出力ディレクトリ**
  完成した動画の保存先です。空欄の場合は「入力動画と同じ場所」に出力されます。
- **一般タブ → オープニング動画／エンディング動画**
  動画の前後に付けたい素材があれば「参照...」で選び、「**結合する**」にチェックを入れます。付けない場合はチェックを外します。
- **一般タブ → 無音カットを有効にする**
  会話のない区間を自動で削るかどうかを選びます。既定はオンです。**「テロップは作りたいが無音カットはしたくない」**場合はチェックを外します（詳しくは[7章](#7-設定項目の説明)参照）。
- **字幕タブ → テロップ生成を有効にする**
  テロップ（字幕）を自動生成したい場合はチェックを入れます。不要ならチェックを外すと、その工程はスキップされます。
- **字幕タブ → 焼き込み前に字幕編集画面を開く**
  自動生成したテロップを、焼き込み前に自分で修正したい場合はチェックを入れておきます（おすすめ）。

各項目の詳しい説明は「[7. 設定項目の説明](#7-設定項目の説明)」を参照してください。
最初はほとんどの項目を初期値のままで問題ありません。

> **重要**: 設定を変更したら、画面下部の **「保存」** ボタンを押すまで反映されません。

## 4. 動画を編集する（基本の流れ）

1. 編集したい動画ファイルを指定します。次のどちらの方法でも構いません。
   - **ドラッグ&ドロップ**: 動画ファイルをメイン画面（「入力動画」欄）へドラッグして放します。**最も手軽な方法です。**
   - **参照ボタン**: メイン画面の **「参照...」** ボタンを押し、ファイル選択ダイアログから選びます。
   （対応形式: `.mp4` `.mov` `.avi` `.mkv` `.flv` `.wmv`）
2. 「入力動画」欄に選んだファイルのパスが表示されていることを確認します。
3. **「実行」** ボタンを押します。
4. あとは画面の案内に沿って操作します（次章参照）。処理が終わると完了メッセージが表示されます。

処理中はステータス欄にくるくる回るマーク（スピナー）と進捗バーが表示され、「動いている」ことが分かります。
音声認識など時間のかかる工程では進捗バーが止まって見えることがありますが、スピナーが回っていれば正常に処理中です。

## 5. 処理中に表示される画面

「実行」を押すと、処理の途中でいくつかの確認画面が自動的に開きます。順番に説明します。

### 5-1. カット閾値の確認（音量解析）

無音カットの前に、アプリが動画の音量を解析し「**音量解析 — カット閾値の確認**」というウィンドウを表示します。
（一般タブで「無音カットを有効にする」をオフにしている場合は、無音カットを行わないため**この確認画面は表示されません**。）

- **この dB（デシベル）以下を無音としてカット**し、それ以上を動画として残します。
- 表示された数値は目安です。必要なら手で修正できます。
  - **もっと多くカットしたい**（無音をしっかり削りたい）→ 数値を **大きく**（例: −28 → −24）
  - **カットしすぎる／会話が切れる** → 数値を **小さく**（例: −28 → −32）
- **「OK」** … 表示（または修正した）値でカットを実行します。
- **「変更しない」** … 既定値のまま処理を続けます（中断ではありません）。

### 5-2. 無音カット・テロップ生成の処理

確認後、次の工程が自動で進みます。

1. **無音カット** … 会話のない区間を除去します。※無音カットを有効にしている場合のみ（オフのときは元の動画がそのまま次の工程へ進みます）
2. **フルテロップ生成** … 音声を認識して字幕（テロップ）を作ります。※テロップ生成を有効にしている場合のみ
   - この工程は動画の長さによって数分〜十数分かかることがあります。スピナーが回っていれば処理中です。

### 5-3. 字幕編集画面（テロップの確認・修正）

設定で「焼き込み前に字幕編集画面を開く」を有効にしている場合、テロップ生成後に「**字幕編集**」ウィンドウが開きます。

| 列 | できること |
|---|---|
| 時間 | テロップの表示時間（編集不可・表示のみ） |
| 字幕 | 認識された文章を修正できます。誤変換をここで直します |
| 使用 | チェックを外した行は動画に焼き込まれません |
| 配信者／サブ／コメント | 行ごとに色（役割）を1つ選べます |

操作のポイント:

- 文章を直すには「字幕」のセルをダブルクリックして編集します。
  - **改行を入れたいとき**: `Alt + Enter` または `Shift + Enter`
  - **編集を確定するとき**: `Enter`（修飾キーなし）
- 下部の **「全て使用」／「全て不使用」** で、チェックを一括で切り替えられます。
- **「字幕決定」** … この内容で動画にテロップを焼き込みます。
- **「キャンセル」** … 処理を中断します（動画は出力されません）。

### 5-4. オープニング・エンディング結合と書き出し

字幕決定のあと、設定に応じてオープニング／エンディングを結合し、動画を書き出します。
完了すると「**処理が完了しました**」というメッセージと保存先のパスが表示されます。

---

## 6. 出来上がった動画の確認

- 完成した動画は、設定画面で指定した **出力ディレクトリ** に保存されます（未指定の場合は入力動画と同じ場所）。
- 完了メッセージに表示されるパスが、実際の保存先です。
- 同じ名前のファイルがある場合は、上書きされないようにファイル名が自動調整されます。

## 7. 設定項目の説明

設定画面（メイン画面の「設定...」ボタン）にある項目の意味です。よく使う項目を中心に説明します。

### 一般タブ

| 項目 | 説明 |
|---|---|
| 動画ディレクトリ | 動画をまとめて置いている既定フォルダ。「参照...」で入力動画を選ぶときの初期表示に使われます |
| オープニング動画 | 本編の前に結合する動画。「結合する」にチェックした場合のみ使われます |
| エンディング動画 | 本編の後に結合する動画。「結合する」にチェックした場合のみ使われます |
| 出力ディレクトリ | 完成動画の保存先。空欄なら入力動画と同じ場所へ出力します |
| 自動更新（起動時に更新を確認する） | 起動時に新しいバージョンを確認します（[9章](#9-アプリの更新について)参照） |
| 無音カットを有効にする | 会話のない区間を自動で削るかどうか。オフにすると無音カット工程をスキップし、元の動画のまま（テロップ生成などの）次の工程へ進みます。※オフのときはカット閾値の確認画面も表示されません |
| フェード | 無音カットのつなぎ目に軽いフェードを付けるかどうか（無音カットが有効なときに使われます） |
| フェード時間(秒) | フェードを付ける場合の長さ（例: `0.05`） |

### 字幕タブ

| 項目 | 説明 |
|---|---|
| テロップ生成を有効にする | テロップ（字幕）を作るかどうか。オフにすると字幕工程をスキップします |
| 焼き込み前に字幕編集画面を開く | 生成したテロップを焼き込み前に確認・修正できます |
| 発話区間のみテロップを表示する | 声のある区間だけにテロップを絞り込みます |
| 発話最小無音長(秒) | この長さ以上の無音で発話区間を区切ります（例: `0.2`） |
| 絞り込み方式 | 「前後の無音を切詰め」／「発話小区間ごとに分割」から選びます |
| 前後パディング(秒) | テロップ表示区間の前後に加える余白（例: `0.1`） |
| 実行デバイス | 音声認識の実行環境（CPU） |
| 計算精度 | 音声認識の精度設定（通常は `int8` のままで問題ありません） |
| 字幕言語 | 認識する言語（`ja` 日本語 / `en` 英語） |
| 配信者カラー／サブカラー／コメントカラー | 役割ごとのテロップ文字色。各行右の色ボタンから選べます |
| フォントの大きさ | テロップの文字サイズ |
| フォント種類 | テロップに使うフォント（PCにインストール済みのものから選択） |
| 改行方式 | 「BudouX(文節)」＝読みやすい位置で改行 ／ 「文字数」＝指定文字数で改行 |
| 1行下限文字数／1行最大文字数 | 1行あたりの文字数の下限・上限 |
| 配信者／サブ／コメント アウトラインカラー | テロップの縁取り色（透明度も指定可） |
| アウトラインの太さ | 縁取りの太さ |
| 背景/影の色 | テロップの背景・影の色（透明度も指定可） |
| 装飾（太字／斜体／下線／打消し線） | テロップ文字の装飾 |
| 文字間隔／回転角度 | テロップの文字間隔と傾き |
| 縁取りスタイル | 「縁取り + ドロップシャドウ」／「不透明ボックス」から選びます |
| 表示位置 | 画面内のテロップ位置（下段中央など） |
| 余白（左／右／下） | テロップと画面端との余白 |

> **色の入力形式について**
> - 配信者／サブ／コメントカラー: `#RRGGBB` 形式（例: `#ED1C24`）
> - アウトライン・背景色: 透明度付きの `&HAABBGGRR` 形式
> いずれも横の色ボタンを押せばカラーピッカーで選べるので、形式を意識せず設定できます。

設定を変えたら、画面下部の **「保存」** を押すのを忘れないでください。

## 8. 困ったときは（トラブルシューティング）

| 症状 | 対処 |
|---|---|
| 「存在する入力動画を指定してください」と出る | 「参照...」で動画を選び直すか、動画ファイルを画面へドラッグ&ドロップしてください。ファイルが移動・削除されていないか確認します |
| ドラッグ&ドロップで動画が入らない | 対応形式（`.mp4` `.mov` `.avi` `.mkv` `.flv` `.wmv`）の動画ファイルを1つだけ放してください。処理の実行中はドロップを受け付けません |
| 無音カットしたくない（テロップだけ作りたい） | 一般タブの「無音カットを有効にする」のチェックを外して保存してください |
| テロップが作られない | 字幕タブの「テロップ生成を有効にする」がオンか確認します |
| 字幕編集画面が出ない | 字幕タブの「焼き込み前に字幕編集画面を開く」をオンにして保存します |
| 無音が削られすぎる／会話が切れる | カット閾値の確認画面で dB を **小さく**（例: −32）してください |
| 無音が残る | カット閾値の確認画面で dB を **大きく**（例: −24）してください |
| テロップの誤変換が多い | 字幕編集画面で文章を直接修正できます |
| 処理が止まって見える | スピナー（回転マーク）が回っていれば処理中です。特に音声認識は時間がかかります |
| エラーで停止した | 表示されたメッセージを確認してください。詳しい記録はインストール先の `logs` フォルダ内のログに残ります |
| 処理を途中でやめたい | 字幕編集画面で「キャンセル」を押すと処理を中断できます（動画は出力されません） |

## 9. アプリの更新について

- 一般タブの「**起動時に更新を確認する**」がオンの場合、アプリ起動時に新しいバージョンがあるか自動でチェックします。
- 新しいバージョンが見つかると「更新の確認」ダイアログが表示されます。
  - **「はい」** … 更新プログラムをダウンロードして自動で適用します（更新中はアプリが再起動されます）。
  - **「いいえ」** … 更新せずにそのまま使います。
- 更新しても、これまでに設定した内容（出力先や字幕の設定など）は引き継がれます。
- 更新の確認を無効にしたい場合は、一般タブの「起動時に更新を確認する」のチェックを外して保存してください。

---

以上で基本的な使い方の説明は終わりです。まずは短い動画で一度試してみて、無音カットの強さやテロップの見え方を確認しながら、設定を自分好みに調整していくのがおすすめです。
