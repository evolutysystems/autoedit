# Stretheus v1.1.0 リリースノート

## 概要
メイン画面をタブ化し、今後の「アーカイブ切り抜き」機能に向けた土台を導入しました（request17 / flow17 R0）。
現行の「クリップ用」自動編集フローは従来どおり動作します（挙動不変・回帰なし）。

## 変更点
- **メイン画面をタブ化**：
  - 「クリップ用」タブ … 現行の自動編集フロー（入力選択・実行・進捗・字幕編集）をそのまま収録。
  - 「アーカイブ切り抜き用」タブ … 準備中（今後のアップデートで採点・切り抜き機能を順次提供）。
- **クリップ用タブの UI 調整**：
  - 「入力動画:」ラベルを廃止（入力欄のプレースホルダで案内）。
  - 「設定」「実行」ボタンをアイコン化（⚙ / ▶）し、横並びに配置。
- **ウィンドウタイトルを「Stretheus」に変更**。

## 互換性
- 既存の `setting.json` はそのまま利用可能（新規キー `archive` は起動時に既定補完）。
- 更新時、ユーザーが設定した出力先・各種設定は保持されます。

## 動作要件
- Windows 11 / 音声認識は CPU 実行（CUDA 不使用）。
- 初回の字幕生成時に音声認識モデルをダウンロードします（ネット接続が必要）。

## 配布
- インストーラー：`StretheusSetup.exe`（オンライン・ダウンローダ型）
- 本体ペイロード：`Stretheus-v1.1.0.zip`（SHA-256 で破損検知）
